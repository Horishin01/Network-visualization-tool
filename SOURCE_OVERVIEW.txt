プロジェクト全体のコード概要（UTF-8テキストのみ）

■ index.html
・役割: ランディングページの HTML エントリ。React の `JS/index.jsx` を読み込む。

■ HTML/training.html / HTML/home.html / HTML/company.html / HTML/about.html
・役割: 各ページの HTML エントリ。対応する React エントリを読み込む。

■ JS/index.jsx / JS/training.jsx / JS/home.jsx / JS/company.jsx / JS/about.jsx
・役割: React エントリ。共通 CSS とページ別 CSS を読み込み、各ページのコンポーネントを描画する。

■ JS/pages/*.jsx
・役割: ページ本体。
  - IndexPage.jsx: 学習スライドの切替。
  - TrainingPage.jsx: 指示書 + 日本地図（改定後レイアウト、Leaflet）。
  - HomePage.jsx: 自宅ネットワーク（パレット/モニター/ルータ設定チュートリアル/外部公開なし）。
  - CompanyPage.jsx: 会社ネットワーク（RT/PCパレット/PC用途切替/サーバアプリ操作/FTPは用意済みHTMLをアップロード）。
  - AboutPage.jsx: サイト概要（用途/技術仕様/ライセンスのまとめ）。

■ JS/components/AppShell.jsx
・役割: 共通ヘッダー/フッター/ナビゲーション。

■ JS/utils/storage.js
・役割: localStorage の load/save ヘルパー。

■ JS/utils/appStore.js
・役割: `app:network:saves:default` の保存/判定（home/company/summary）。

■ CSS/style.css
・役割: 共通レイアウト（罫線背景、ヘッダー、フッター、パネル類）。

■ CSS/index.css / CSS/training.css / CSS/home.css / CSS/company.css / CSS/about.css
・役割: ページ固有のレイアウト。
