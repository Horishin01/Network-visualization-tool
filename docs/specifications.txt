Network Visualization Tool - 仕様メモ（PDF準拠）
日付: 2026-01-05 (JST)
※生成AI運用規定: docs/ai_usage_guidelines.txt を必ず参照

概要
- 研究仕様書（PDF）に沿ったレイアウトを React で再構成する。
- ページ構成は index / training / home / company / about。

レイアウト要点
- 共通: ヘッダー（ホーム/トレーニング/このサイトについて）+ フッター。
- index: スライド枠（学習内容の説明）と前後ボタンで切替。
- training: 左に指示書、右に日本地図（改定後レイアウト）。右カラムのプレビュー欄は削除。
  - 日本地図は main ブランチと同じ Leaflet 表示（OpenStreetMap）。
- home: 左に指示書、中央に自宅シミュレータ（家 + ONU）、右に機器パレットとモニター。
- company: 左に指示書、中央に会社シミュレータ（建物 + ONU）、右に機器パレットとモニター。

要素（PDF「要素」表より）
- I-学習説明: index のスライド内容。
- I-スライド切替: クリックでスライドを切替。
- T-全体表示: 進行や次のステップを示す注記。
- T-日本地図: home/company をクリックで遷移。
- H/C-指示書: 各ページの説明欄。
- H/C-機器パレット: クリックで機器追加。
- H/C-シミュレータ: 追加機器の表示と簡易設定の表示。

保存
- 機器配置や選択状態は localStorage に JSON 保存。
- 判定結果は `app:network:saves:default` に保存（home/company/summary）。

使用技術
- JavaScript / HTML / CSS / React（Vite でビルド）
- Leaflet（OpenStreetMap）
